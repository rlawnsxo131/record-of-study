# 5장 문

```
거의 대부분 생략.
자바스크립트를 어느정도 사용하던 사람이면 그냥 술술 훑어도 될듯하다.
관련된 몇몇 키워드의 깊은 내용은 뒷장에 서술한다고 한다.
```
